**Why I made this**
<br>
I built this project because I genuinely miss the old physical photo albums 
our parents used to make.
<br>
Nowadays, our memories are all digital, scattered across clouds, 
lost between screenshots, and buried under thousands of other files. 
<br>I wanted a space where photos felt special again. Not just floating images, 
but a connected ribbon of moments that feels tactile and intentional.

## Features âœ¨
* **Fluid 3D Scroll:** Scroll through time with smooth physics.
* **Handwritten Feel:** Uses a handwriting font (Caveat) to feel like a personal diary.
* **Auto-Sorting:** Automatically arranges your memories chronologically, no matter when you add them.
* **Private & Local:** Your photos and notes are saved directly to your browser's Local Storage.

## How to run
You can view the live version here: **[[My Memory Ribbon](https://mariamanbar.github.io/memory-ribbon/)]**

Or run it locally:
1.  Clone the repo.
2.  Open `index.html` in any browser.

* HTML5
